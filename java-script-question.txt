После var a = {a: { a1 : 1 }, b: "string" }, c = a, b = { a : a.a, b: a.b }; delete a.a;

b.a будет содержать { a1: 1 }, ибо b ссылается на deep-copy объекта a, посему посему не зависим от a.
c.a будет содержать undefined, ибо c ссылается на тот же объект, что и a, посему отражает факт удаления поля a через a.
